# Danger Mage

A script for Superfighters Deluxe

This is a complex game modification that adds magic spells to SFD through the use of the chat bar.
typing any combination of letters will produce a unique spell combination which you can then cast by flicking your wand.

This script will be available on [https://sfdmaps.at.ua/](https://sfdmaps.at.ua/) as well as the steam workshop for the game.
In order to compile the source code you need Shutdownman's class welder ([download link](https://mega.nz/file/4AMWlR6L#kSXMjdYtuuVle6x8LU7aYUiY3bu8vHWGjT_JxGi-9uY)) or NotBot's [Script Util](https://github.com/NotRustyBot/scriptutil).
